## install

* 安装node.js环境 

因为要使用npm命令去安装软件,因此需要安装Node.js
去官网https://nodejs.org/en/下载

* 安装hexo

```js
	npm install hexo-cli -g
```

* 从git上拉取代码

```js
	git clone git@github.com:hlhdidi/mynote.git
```

* 启动hexo

```js
	cd mynote
	hexo server
```
